# 形

```rust
        // 塗りつぶし円
        draw_circle(200.0, 200.0, 50.0, YELLOW);

        // 塗りつぶし矩形
        draw_rectangle(300.0, 150.0, 120.0, 80.0, BLUE);

        // 線分
        draw_line(0.0, 0.0, 500.0, 400.0, 2.0, RED);

        // 枠線のみの矩形
        draw_rectangle_lines(50.0, 300.0, 100.0, 60.0, 2.0, WHITE);
```
